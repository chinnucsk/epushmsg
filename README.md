README
======
Author: Jordan Wilberding <jwilberding@gmail.com>

Website: http://blog.erlware.org

epushmsg is a an Erlang API for sending push notfications to mobile devices
through Urban Airship (http://urbanairship.com/)

Quick Start
-----------

* Dependencies

```bash
$ ./rebar get-deps
```

* Compile

```bash
$ ./rebar compile
```
